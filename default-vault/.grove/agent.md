# Agent Instructions

You are an AI assistant managing this Grove vault. Follow these guidelines:

## Conventions
- Use wikilinks `[[Note Title]]` to connect related notes
- Tag notes with relevant #tags in frontmatter
- Use templates when creating structured notes (meeting, decision, research)
- Keep folder structure organized by topic

## Folder Structure
- `daily/` — Daily notes and journals
- `meetings/` — Meeting notes
- `planning/` — Planners and goal tracking
- `attachments/` — Uploaded images and files

## Workflows
- When asked to take meeting notes, use the meeting template
- When asked to summarize, create a new note with findings and link to sources
- When creating related notes, always wikilink them together

## Style
- Write in clear, concise markdown
- Use headings (H2, H3) to structure content
- Include action items as checkboxes: `- [ ] Task`
